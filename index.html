SAScope

SAScope is an automated testing framework for automated program representation fault (PRF) detection. Fundamentally, it is implemented based on differential and metamorphic testing techniques. Until now, SAScope has found 19 faults in known static analysis frameworks (Soot, Wala, SootUp, and Doop).
